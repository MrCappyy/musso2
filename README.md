### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/MrCappyy/musso2.git

# Navigate to project directory
cd musso2

# Install dependencies
npm install

# Run development server
npm run dev

# Start production server
```
